# Licensing Overview

This repository is source-available software. It is not offered under an
OSI-approved open-source license.

## Default license

Unless a file states otherwise, the software is licensed under:

```text
PolyForm Noncommercial License 1.0.0
SPDX identifier: PolyForm-Noncommercial-1.0.0
```

The authoritative license text is stored in:

```text
LICENSE
LICENSES/PolyForm-Noncommercial-1.0.0.txt
```

The duplicate under `LICENSES/` exists for SPDX- and REUSE-compatible tooling.
The root `LICENSE` file remains present for GitHub and ordinary repository
browsing.

## Additional hobbyist permission

`ADDITIONAL_PERMISSION.md` grants a narrow Five-Unit Cost-Recovery Permission to
natural persons who satisfy every condition in that document.

The permission is additional to the base license. It does not convert the
project into open-source software and does not grant general commercial rights.

## Contributions

External code contributions are currently not accepted. This avoids ambiguous
copyright ownership and preserves the ability to offer separate commercial
licenses.

Bug reports, design discussions, documentation corrections, and reproducible
test cases remain welcome. See `CONTRIBUTING.md`.

## Third-party material

Third-party code, generated files, examples copied from external sources, and
dependencies must retain their original notices and licenses. They must not be
silently relicensed under the project license.

A contribution or dependency must not be added when its license is incompatible
with the project's intended distribution and commercial-licensing model.

## No legal advice

This document is a project overview, not legal advice. The authoritative rights
and obligations are defined by the license and permission texts themselves.

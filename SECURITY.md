# Security Policy

Eurorack Framework is embedded software and may control physical hardware. Bugs can result in unexpected voltages, signal levels, timing, device behavior, or hardware stress.

Report security-sensitive or safety-relevant issues privately to:

eurorack@skjt.de

Do not include confidential hardware designs, credentials, private keys, or personal information in public issues.

The project is provided without warranty. Users remain responsible for validating electrical limits, protection circuits, calibration, and safe behavior on their target hardware.
